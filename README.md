# iHoogi - מצגת למשרדי פרסום

מצגת אינטראקטיבית בעברית (RTL) עבור משרדי פרסום ושיווק דיגיטלי.

## התקנה והרצה

```bash
npm install
npm run dev
```

## בנייה

```bash
npm run build
```

## הוספת הלוגו

1. שמור את תמונת הלוגו (ינשוף-רובוט) בתיקייה `public/images/` בשם `logo.png`
2. הלוגו יופיע אוטומטית בכל עמוד עם הטקסט "ihoogi — העוזר שמסנן פניות בשבילך"
3. אם התמונה לא קיימת, יופיע fallback עם האותיות "iH"

## שקפים (9 שקפים)

המצגת כוללת 9 שקפים מלאים:

1. **הכאב** - הרבה לידים ואין שליטה אמיתית
2. **לידים חכמים** - כל ליד מקבל ציון (חם/פושר/קר)
3. **לידים חמים לנייד** - מגיעים ישירות עם תשובות הלקוח
4. **מוכנות למכירה** - יודעים מה הלקוח רוצה עוד לפני השיחה
5. **שאלון חכם** - נבנה אוטומטית ע"י המערכת
6. **תגובה חכמה** - מענה אוטומטי מותאם ללקוח
7. **ניתוח קמפיינים** - יודעים איזה קמפיין מביא לקוחות
8. **זמן ושליטה** - יותר סדר, פחות רעש
9. **CTA** - התנסות חינמית לזמן מוגבל

## ניווט

- חצים ← → או ↑ ↓ למעבר בין שקפים
- לחיצה על הנקודות בתחתית לקפיצה לשקף ספציפי

## הוספת תמונות

בכל שקף שמכיל `imagePlaceholder`, החלף את ה-placeholder בתמונה המתאימה:
- שמור תמונות ב-`public/images/`
- עדכן את `imagePlaceholder` ב-`src/App.tsx` עם נתיב התמונה

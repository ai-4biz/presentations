# תמונות - הוספת הלוגו שלך

## ⚠️ חשוב: שמרי את הלוגו שלך כאן!

1. **לחצי לחיצה ימנית** על התמונה של הלוגו שלך (הינשוף-רובוט)
2. בחרי **"שמור תמונה בשם..."** או **"Save image as..."**
3. שמרי את הקובץ בתיקייה הזו בשם אחד מהשמות הבאים:
   - `logo.png` (מומלץ)
   - `logo.jpg`
   - `logo.jpeg`
   - `logo.webp`

## איפה התיקייה?
```
public/images/
```

## אחרי שמירת התמונה:
- רענני את הדפדפן (F5)
- הלוגו שלך יופיע אוטומטית בכל עמוד!

## אם הלוגו לא מופיע:
1. ודאי שהקובץ נקרא בדיוק `logo.png` (או `logo.jpg`)
2. ודאי שהקובץ נמצא בתיקייה `public/images/`
3. רענני את הדפדפן (Ctrl+F5)

# איך להשתמש בקובץ המצגת לוואטסאפ

## הקובץ שנוצר
`ihoogi-presentation-standalone.html` - קובץ HTML אחד שכולל את כל המצגת

## איך לשלוח בוואטסאפ:

### אפשרות 1: שליחה ישירה (מומלץ)
1. פתחי את הקובץ `ihoogi-presentation-standalone.html` בדפדפן
2. לחצי על "שלח" או "Share" בדפדפן
3. בחרי "WhatsApp" או העתיקי את הקישור
4. שלחי את הקישור בוואטסאפ

### אפשרות 2: העלאה לאינטרנט
1. העלי את הקובץ לשירות אירוח כמו:
   - GitHub Pages (חינם)
   - Netlify (חינם)
   - Vercel (חינם)
   - Google Drive (חינם)
2. שלחי את הקישור בוואטסאפ

### אפשרות 3: שליחה כקובץ
1. שלחי את הקובץ `ihoogi-presentation-standalone.html` בוואטסאפ
2. המקבל יפתח אותו בדפדפן

## הוספת הלוגו

הקובץ מחפש את הלוגו באותה תיקייה:
- `logo.png` או `logo.jpg`

אם את שולחת את הקובץ, ודאי שגם הלוגו באותה תיקייה.

## תכונות הקובץ:
✅ עובד ללא אינטרנט (לאחר טעינה ראשונית)
✅ כולל את כל 9 השקפים
✅ ניווט עם מקלדת (חצים) או עכבר
✅ עיצוב מלא עם כל הצבעים והסגנונות
✅ Responsive - עובד על מובייל ומחשב

## פתרון בעיות:
- אם הלוגו לא מופיע: ודאי ש-`logo.png` באותה תיקייה
- אם הסגנונות לא עובדים: ודאי שהקובץ נפתח בדפדפן מודרני (Chrome, Firefox, Safari, Edge)

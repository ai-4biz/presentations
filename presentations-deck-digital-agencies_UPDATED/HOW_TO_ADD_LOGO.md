# איך להוסיף את הלוגו שלך

## שלב 1: שמירת התמונה

1. לחצי על התמונה ששלחת (הינשוף-רובוט)
2. לחצי על "שמור תמונה בשם..." או "Save image as..."
3. שמרי את התמונה בתיקייה:
   ```
   public/images/logo.png
   ```
   או
   ```
   public/images/logo.jpg
   ```

## שלב 2: וידוא שהקובץ נשמר

הקובץ צריך להיות בנתיב:
- `public/images/logo.png` 
- או `public/images/logo.jpg`
- או `public/images/logo.svg`

## שלב 3: רענון הדפדפן

אחרי שמירת התמונה, רענני את הדפדפן (F5) והלוגו יופיע אוטומטית בכל עמוד!

---

**טיפ:** אם התמונה לא מופיעה:
1. ודאי שהקובץ נקרא בדיוק `logo.png` (או `logo.jpg`)
2. ודאי שהקובץ נמצא בתיקייה `public/images/`
3. רענני את הדפדפן

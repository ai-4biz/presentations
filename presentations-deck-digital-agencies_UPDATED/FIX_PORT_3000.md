# פתרון לבעיית פורט 3000 תפוס

## הבעיה
פורט 3000 תפוס על ידי תהליך אחר (כנראה Vite שרץ מהפעלה קודמת).

## פתרונות:

### פתרון 1: סגירת התהליך הישן (מומלץ)
1. פתחי Task Manager (Ctrl+Shift+Esc)
2. חפשי תהליך בשם "node" או "vite"
3. סגרי את התהליך
4. הרצי שוב: `npm run dev`

### פתרון 2: שימוש בפורט אחר
הקוד כבר מוגדר לעבור אוטומטית לפורט אחר אם 3000 תפוס.
אם 3000 תפוס, השרת יעלה על 3001, 3002 וכו'.

### פתרון 3: שינוי ידני של הפורט
אם את רוצה פורט ספציפי אחר, עדכני את `vite.config.ts`:
```typescript
server: {
  port: 3001, // או כל פורט אחר
  strictPort: true
}
```

## בדיקה מהירה
הרצי: `npm run dev`
אם תראי "Port 3000 is in use, trying another one..." - זה אומר שהקוד עובד נכון ועבר לפורט אחר.
